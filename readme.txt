"WooCommerce Pay What You Want" lets users name their own price

== Description ==

The WooCommerce Pay What You Want extension lets you be flexible in
what price you are willing to accept for selected items by letting
clients decide what to pay for simple & variable products.

== Installation ==
1. Upload or extract the `woocommerce-pay-what-you-want` folder to your site's 
   `/wp-content/plugins/` directory. You can also use the *Add new* 
   option found in the *Plugins* menu in WordPress.  
2. Enable the plugin from the *Plugins* menu in WordPress.
3. In WooCommerce -> `Pay What You Want` to enable the options.

= Usage =

1. Install the "WooCommerce Pay What You Want" Plugin.
2. Activate the plugin in the plugin page.
2. Activate the sitewide by going to WooCommerce -> `Pay What You Want` .
3. Go to the WooCommerce Product Page you want to allow clients to set the price for themselves.
4. Set to enable client pricing.
5. Set mininum, maximum, and recommended prices.
6. Set to display the prices for each of these.
